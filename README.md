# 🚒 제7회 대구 빅데이터 분석 경진대회

> **대구시 소방차 출동로 취약지역 분석 및 머신러닝 기반 위험도 예측 및 정책 시뮬레이션**

---

## 🏆 대회 개요
- **대회명:** 제7회 대구 빅데이터 분석 경진대회  
- **주최:** 대구광역시 × 대구디지털혁신진흥원 / 데이콘  

### 📌 문제 정의
대구시 읍면동 단위에서 **화재 출동 도착시간과 출동횟수를 예측**하고,  
이를 기반으로 **출동로 취약지역을 도출하고 정책 개선 효과를 분석**하는 것을 목표로 합니다.  

단순히 위험 지역을 식별하는 것을 넘어,  
**왜 위험한지 설명하고, 개선 시 얼마나 효과가 있는지를 정량적으로 제시하는 것**을 핵심 목표로 설정했습니다.

---

## 🎯 분석 접근

본 프로젝트는 단순 예측 모델 구축이 아니라,  
실제 정책 의사결정에 활용 가능한 구조를 만드는 것을 목표로 설계했습니다.

- 어디가 위험한가 → **예측**
- 왜 위험한가 → **SHAP 기반 해석**
- 무엇을 바꾸면 얼마나 개선되는가 → **정책 시뮬레이션**

이 세 가지 흐름을 하나의 파이프라인으로 연결하여  
**예측 → 해석 → 정책 적용**까지 이어지는 구조를 구성했습니다.

## 🔄 분석 파이프라인

1. RAW 데이터 수집

---

## 🧩 데이터 처리

- 행정동 기준으로 다양한 출처의 데이터를 통합하여 분석 단위를 일관되게 구성  
- 결측치: 인구, 면적, 인구밀도 등 지역 특성을 반영한 가중 평균 방식으로 보정  
- 이상치: 도착시간 60분 이상 데이터 제거하여 모델 왜곡 방지  
- 범주형 변수: One-Hot Encoding 적용  
- 타깃 변수: log 변환 후 예측값은 역변환하여 해석 가능하도록 구성  

단순 전처리를 넘어, 각 변수의 의미와 관계를 고려하여  
**데이터의 신뢰도를 높이는 방향으로 전처리를 설계**했습니다.

---

## 🧠 분석 방법

### 🔹 데이터 분리 전략

화재 출동은 시간적 요인의 영향을 크게 받기 때문에  
하나의 통합 모델로 접근하기보다, 다음과 같이 데이터를 분리하여 분석했습니다.

- 시간별  
- 요일별  
- 월별  

각 단위별로 데이터 특성이 다르기 때문에  
**독립적인 모델을 구성하여 보다 정밀한 패턴을 학습하도록 설계**했습니다.

---

## 🤖 모델

- 모델: XGBoost Regressor  
- 검증: KFold 교차검증  
- 튜닝: Optuna 기반 하이퍼파라미터 최적화  
- 해석: SHAP 기반 변수 중요도 분석  

👉 총 6개 모델 구성  
(시간 / 요일 / 월 × 출동횟수 / 도착시간)

XGBoost는 다양한 변수 간 비선형 관계를 효과적으로 학습할 수 있고,  
SHAP과 결합하여 **예측 결과를 해석 가능하게 만든다는 점에서 적합한 모델**이라고 판단했습니다.

---

## 🔍 SHAP 분석

- 지역별 주요 영향 변수 도출  
- 동일 위험지역 간 원인 차이 분석  
- 정책 개선 방향 도출  

예측 결과를 단순 수치로 끝내지 않고,  
**“왜 해당 지역이 위험한지”를 설명 가능한 형태로 확장**했습니다.

이를 통해 지역별 맞춤형 정책 설계가 가능하도록 기반을 마련했습니다.

---

## 🧪 정책 시뮬레이션

- 도로폭, 협소도로 비율 등 정책적으로 개선 가능한 변수 조정  
- 변화 전 / 후 예측값 비교  

👉 단순 분석이 아닌  
**정책 적용 시 효과를 정량적으로 검증하는 단계까지 확장**

이를 통해 분석 결과를 실제 의사결정에 활용할 수 있는 형태로 연결했습니다.
